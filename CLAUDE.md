# CLAUDE.md — 局域网 IP 扫描工具

## 项目概述

这是一个 Windows 局域网 IP 扫描工具。用户双击 `启动扫描工具.bat` 后，自动在浏览器中打开扫描页面，可扫描本机所在局域网内所有存活设备，识别设备类型（手机/电脑/打印机/路由器等），并汇总空闲 IP。

## 技术栈

- **后端**：Python 3.11 + Flask
- **前端**：纯 HTML/CSS/JS（单文件）
- **扫描**：系统 ping + ARP 表查询 + MAC 厂商识别
- **运行方式**：批处理文件启动 Flask 服务 → 自动打开浏览器

## 注意事项

- 本工具仅在 Windows 上运行，依赖系统 `ping` 命令
- 扫描需要访问局域网，首次运行时 Windows 防火墙可能弹窗，需允许
- 不支持跨子网扫描，仅扫描本机所在 /24 网段
